# EduTrack

A mobile app to help students track their school assignments, upcoming exams, and school projects.

## Tech Stack

- **Platform**: Compose Multiplatform (Android + iOS)
- **Package**: `com.edutrack`
- **Backend**: Google Sheets (CMS)
- **UI**: Material 3 (Jetpack Compose)
- **DI**: Koin
- **Networking**: Ktor
- **Architecture**: MVI (Model-View-Intent) with Clean Architecture

## Features

- **Assignment Tracker** — View teacher-managed assignments and track personal completion status with reminders
- **Exam Planner** — View upcoming exams with study topics and receive reminders
- **Project Manager** — View school projects with milestones and receive reminders
- **Dashboard** — Overview of upcoming assignments, exams, and projects
- **Onboarding** — Welcome walkthrough for first-time users
- **Settings** — App preferences including Google Sheet configuration and dark mode

## Getting Started

### Prerequisites

- Android Studio Hedgehog or later
- Xcode 15+ (for iOS)
- JDK 17+

### Build & Run

**Android:**
```bash
./gradlew :composeApp:assembleDebug
```

**iOS:**
```bash
cd iosApp
pod install
open iosApp.xcworkspace
```
