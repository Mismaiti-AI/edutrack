# EduTrack

An app to help students track their school assignments, upcoming exams, and school projects.

## Tech Stack

- **Platform**: Compose Multiplatform (Android + iOS)
- **Package**: `com.edutrack`
- **Backend**: Google Sheets (CMS)
- **UI**: Material 3 (Jetpack Compose)
- **DI**: Koin
- **Networking**: Ktor
- **Architecture**: MVI (Model-View-Intent) with Clean Architecture

## Features

- **Assignment Tracking** — View and track school assignments from teacher's Google Sheet
- **Exam Tracking** — View upcoming exams with study references from teacher's Google Sheet
- **Project Tracking** — View school projects with team info from teacher's Google Sheet
- **Onboarding** — First-time user walkthrough with 3 swipeable pages
- **Settings** — App preferences including dark mode toggle and Google Sheet configuration

## Getting Started

### Prerequisites

- Android Studio Hedgehog or later
- Xcode 15+ (for iOS)
- JDK 17+

### Build & Run

**Android:**
```bash
./gradlew :composeApp:assembleDebug
```

**iOS:**
```bash
cd iosApp
pod install
open iosApp.xcworkspace
```
